# OxTech Academy

Landing page for OxTech Academy — practical AI development, prompt engineering, and digital skills training for African professionals, students, and career-switchers.

## Structure

```
index.html      Single-page site (nav, hero, about, services, contact, footer)
css/style.css   Styles
js/script.js    Nav toggle, scroll animations, contact form demo handler
assets/         Images
```

## Development

No build step — open `index.html` directly in a browser, or serve it locally:

```
npx serve .
```
